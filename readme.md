# Build a Technical Documentation Page

A responsive web design project for the Responsive Web Design Certification in [freeCodeCamp.org](https://www.freecodecamp.org/learn/)

Happy coding!

## Specifications or User Story

1. `main#main-doc`
2. Minimum of 5 `section.main-section`
3. `header` inside each `section`
4. ID in `header#{topic_underscore_separated}`
5. Minimum of 10 `p` total in `section.main-section`
6. Minimum of 5 `code` total in `section.main-section`
7. Minimum of 5 `li` total in `section.main-section`
8. `nav#navbar`
9. `header` inside `nav#navbar`
10. `ul li a.nav-link` inside `nav#navbar` and `section.main-section`
11. `header` before `a` in `nav#navbar`
12. Text of `a.nav-link` corresponds the `section header`
13. `nav a.nav-link` click navigates to the corresponding section of the `main#main-doc`
14. Responsive layout of `nav#navbar` shown on the left side of the screen for regular sized devices
15. Should have atleast 1 media query
